# Cybotic System UI

## 義體改造管理系統

一個在 Godot 中實現的先進義體改造介面，融合了俄羅斯方塊的放置機制。

### 功能特性

#### 🎮 核心系統
- **動態格子系統**：根據義體部位自動調整格子大小（1x1 到 8x8）
- **模組放置**：使用俄羅斯方塊形狀的模組進行義體增強
- **旋轉機制**：支援模組在格子內旋轉（按 R 鍵）
- **拖放操作**：支援滑鼠拖拽移動已放置的模組
- **無消除機制**：放置的模組不會自動消失

#### 📊 模組類型

| 類型 | 顏色 | 效果 | 描述 |
|------|------|------|------|
| 冷卻型 | 淺藍 | ↓熱量 ↑散熱 | 降低熱量產生，提升散熱效率 |
| 超載型 | 紅色 | ↑輸出 ↑熱量 | 增加能力輸出，但提高熱量產生 |
| 能源型 | 黃色 | ↑供能 ↑容量 | 增加能源供應和耗電管理 |
| 防禦型 | 綠色 | ↑防禦 ↑耐久 | 提升義體防禦和耐久性 |
| 處理型 | 紫色 | ↑計算 ↑速度 | 提升計算能力和反應速度 |
| 強化型 | 橙色 | ↑全能 | 全面強化義體結構 |

#### 🤖 義體部位

- **頭部** (3×3 格子)：感測和處理系統
- **軀幹** (8×8 格子)：主要能源和冷卻中樞
- **左臂** (2×6 格子)：左側增強模組
- **右臂** (2×6 格子)：右側增強模組
- **左腿** (2×6 格子)：左側增強模組
- **右腿** (2×6 格子)：右側增強模組

### 使用說明

#### 基本操作

1. **選擇身體部位**：在左側人型視圖中點擊要改造的部位
2. **選擇模組類型**：在右側選擇所需的模組類型
3. **選擇俄羅斯方塊形狀**：點擊按鈕選擇形狀（I, O, T, S, Z, J, L）
4. **放置模組**：在中央格子中左鍵點擊放置模組
5. **移動模組**：左鍵拖拽已放置的模組移動位置
6. **旋轉模組**：選中模組後按 R 鍵旋轉
7. **刪除模組**：選中模組後按 Delete 鍵移除

#### 快捷鍵

- `R` - 旋轉已選中的模組
- `Delete` - 刪除已選中的模組
- `左鍵點擊` - 選擇或放置模組
- `左鍵拖拽` - 移動模組

### 屬性系統

#### 基礎屬性

- **熱量產生** (°C/s)：義體產生的熱量
- **冷卻效率** (%)：冷卻系統的工作效率
- **耗電量** (W)：義體耗電功率
- **能源供應** (W)：能源系統輸出功率
- **能量容量** (kJ)：能量儲存容量
- **輸出功率** (%)：能力輸出強度
- **防禦力** (pts)：防禦能力
- **耐久性** (pts)：結構耐久程度
- **處理能力** (GFLOPS)：計算處理速度
- **反應速度** (ms)：反應時間
- **整體效率** (%)：系統總體效率
- **重量** (kg)：義體總重量

#### 特殊能力

系統根據屬性組合自動觸發特殊能力：

- **高效散熱**：冷卻效率 > 150%
- **強力供能**：能源供應 > 100W
- **超載輸出**：能力輸出 > 50%
- **高速計算**：計算能力 > 50GFLOPS
- **堅固外殼**：防禦力 > 50pts
- **整體優化**：系統效率 > 130%
- **過熱警告** ⚠️：熱量產生 > 100°C/s

### 技術架構

#### 系統分層

1. **常數層** (`constants.gd`)
   - 定義所有常數、枚舉和配置
   - 模組類型、顏色、屬性效果映射

2. **工具層** (`tetromino_shapes.gd`)
   - 俄羅斯方塊形狀定義和操作
   - 旋轉、碰撞檢測

3. **系統層**
   - `module_system.gd`：模組數據管理
   - `stats_calculator.gd`：屬性計算引擎
   - `cyborg_manager.gd`：統一協調器

4. **UI層**
   - 各個UI組件負責渲染和互動
   - `main_ui.gd`協調所有UI組件
